
# Вывод информации об IP с помощью API в CLI

***
### Требования
Для запуска проекта вам понадобятся следующие компоненты:

- Go 1.21.1

***

1. Склонируйте репозиторий на вашу локальную машину:

   ```bash
   git clone https://github.com/Jhnvlglmlbrt/ip-tracker-cli

2. Перейдите в директорию проекта:

   ```bash
   cd ip-tracker-cli

3. Установите зависимости проекта:

    ```bash
    go get
    
4. Вместо api вставьте свой ключ:

    ```bash
    make run

4. Запустите проект: 

    ```bash
    ./bin/app trace 1.1.1.1 2.2.2.2
    либо
    ./bin/app version
